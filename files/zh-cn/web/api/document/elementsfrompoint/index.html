elementsFromPoint() 是 {{domxref("DocumentOrShadowRoot")}} 下的一个函数，该函数返还在特定坐标点下的HTML元素数组。

语法

var elements = document.elementsFromPoint(x, y);

参数

x
坐标点的水平坐标值
y
坐标点的垂向坐标值

返回值

一个包含 {{domxref('element')}} 对象的数组.
